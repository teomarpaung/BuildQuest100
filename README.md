# 🚀 Python BuildQuest – 100 Projects in 100 Days

Selamat datang di repository **Python BuildQuest**!
Ini adalah perjalanan saya untuk menguasai Python dengan membangun **100 proyek nyata dalam 100 hari** berdasarkan e-book *Python BuildQuest*.

> 🔥 Fokus utama: belajar dengan praktik, bukan hanya teori.

---

## 🎯 Tujuan Proyek

* Menguasai Python dari dasar hingga lanjutan
* Membangun 100 proyek nyata (real-world projects)
* Membuat portofolio publik untuk karier / freelance
* Melatih problem solving seperti developer profesional

---

## 🧭 Struktur Perjalanan (100 Hari)

Repository ini dibagi menjadi 5 fase utama:

### 🟢 Fase 1 – Dasar Python & Logika (Hari 1–10)

* Sintaks dasar Python
* Variabel, input/output
* If-else, loop, fungsi
* Program sederhana (Hello World, kalkulator, dll)

📁 Folder: `phase1_basic/`

---

### 🟡 Fase 2 – Struktur Data & Otomatisasi (Hari 11–30)

* Struktur Data
* Automasi tugas sederhana
* Proyek seperti:

  * Auto-grader
  * To-do list
  * Kalkulator lanjutan

📁 Folder: `phase2_struktur_data/`

---

### 🔵 Fase 3 – Web & API (Hari 31–50)

* HTTP request & API
* Web app sederhana dengan Flask
* Proyek:

  * Weather app
  * Quote generator
  * REST API mini

📁 Folder: `phase3_web_api/`

---

### 🟣 Fase 4 – Data & AI (Hari 51–80)

* Analisis data (pandas)
* Visualisasi (matplotlib)
* Machine Learning dasar

📁 Folder: `phase4_data_ai/`

---

### 🔴 Fase 5 – Final Project (Hari 81–100)

* Integrasi semua skill
* Membangun produk nyata
* Deploy & dokumentasi

📁 Folder: `phase5_final_project/`

## ⚙️ Cara Menjalankan Proyek

1. Clone repository:

```bash
git clone https://github.com/username/BuildQuest100.git
cd BuildQuest100
```

2. Jalankan file Python:

```bash
python nama_file.py
```

3. (Opsional) Gunakan virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```
